# Image Manifest — Peru Reis 2026

## How to use this file

For each image slot below, fill in the **Jouw foto** column with either:

- A **local file path**: `images/huaraz/hero.jpg` ← drop your photo in the `images/` folder
- An **external URL**: `https://...` ← paste a URL from any photo site

Leave it blank if you have nothing yet. Once you've filled in what you have,
tell Claude _"update the website from IMAGES.md"_ and all HTML files will be updated in one pass.
